# SQL-Film-Rental-Business-Analysis

## Project Overview
This project analyzes a film rental business database using SQL to extract meaningful business insights.
The analysis focuses on customer behavior, rental trends, and revenue distribution to support data-driven decisions.

## Tools Used
- SQL (MySQL)
- Sample Film Rental Database (Sakila)

## Key SQL Skills Applied
- Data filtering, sorting, and conditional querying (SELECT, WHERE, ORDER BY)
- Data aggregation and grouping using GROUP BY and HAVING
- Multi-table analysis using INNER and LEFT JOINs
- Subqueries for advanced filtering and comparison
- Use of aggregate functions such as COUNT, SUM, and AVG

## Project Objective
- Analyze rental and payment data
- Identify top-performing films and customers
- Evaluate store-wise revenue and rental activity

## Key Insights
- A small set of films contributes to a large share of total rentals.
- Repeat customers generate a significant portion of overall revenue.
- Revenue and rental activity vary across different stores.
- Certain film categories consistently show higher rental demand.
- Rental patterns indicate time-based trends in customer behavior.
